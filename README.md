Mobile-First Website with Multi-Language Support

This repository contains the code for a mobile-first, multi-language website built for my company. The website is developed using React and Next.js, and features a custom CMS to allow for easy content updates without the risk of damaging the site or causing errors.

Features:

- Main Website in German with an English version synced via Canonical and hreflang tags. The German version is located under .de and the English version under .com.

- Fully Responsive Design optimized for mobile-first approach.

- SEO & GEO Optimization for better search engine ranking and localized search performance.

- Start Video that spans the full width of the homepage for an immersive experience.

- Discrete Animations and smooth scroll effects for a dynamic user experience.

- Image Slideshow to highlight key content or promotions.

- Google Reviews integration, showing star ratings and feedback.

- Client Logos showcasing companies and organizations with which we have collaborations.

- Two Email Integrations:

- One for initial contact.

- One for communication with existing customers.

Technologies Used:

- React & Next.js for the front-end.

- Custom CMS for easy content management and updates.

- SEO and GEO Optimization for better visibility.

- Google Reviews API for displaying ratings.

- CSS Animations for smooth transitions and scroll effects.
